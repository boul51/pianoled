Libraries needed to build the sketch :

MIDIUSB 1.0.2 (https://www.arduino.cc/en/Reference/MIDIUSB)
Arduino midi library 4.3.1 (https://github.com/FortySevenEffects/arduino_midi_library)
Adafruit Neopixel 1.0.6 (https://github.com/adafruit/Adafruit_NeoPixel)

Utility used for the serial to MIDI bridge under linux :
ttymidi (from https://github.com/robelix/hard-dj.git)
